# Functions Challenge

A JavaScript practice repo focused on functions and arrays. The files contain small coding challenges with expected console output.

## Topics Covered

- String interpolation
- Even/odd checks
- Celsius to Fahrenheit conversion
- Finding maximum values
- `setTimeout`
- Array duplicate removal
- Filtering even numbers
- Array shifting and rotation

## Files

```text
index.html
functions-part1.js
functions-part2.js
arrays.js
Part1.md
Part2.md
Arrays.md
```

## How To Run

Open `index.html` in a browser and check the browser console.

## What I Practiced

- Writing reusable functions
- Returning values
- Using array helper methods
- Reading expected output comments
- Debugging with `console.log`
